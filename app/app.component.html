About

This webapp is an interface to one of NASA's public APIs. I, Marc Normandin, wrote it while learning Angular 2 in 2016. It is written in Angular 2, and uses Bootstrap for styling. It is served by an nginx server.

Some known issues are that some images are returned with junk data, and so will be displayed as pure white. This seems to be a problem with the googleapis server. The images are requested in sequence with 1 second delays because the googleapis server will give a 429 response (Too Many Requests) otherwise. The images are not sorted by date before being displayed, and this will be fixed in the future.

To-do: I intend to host the source code for this web app on GitHub in the future. This is not currently done because I need to change the architecture a bit so that the token is not exposed. In addition, the client will use a custom node.js server to get images from instead of contacting the NASA directly.
